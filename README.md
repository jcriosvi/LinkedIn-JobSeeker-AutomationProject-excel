# LinkedIn-JobSeeker-AutomationProject-excel
It allows to perform a job search based on rol and company name and the results are store in excel file (job url, rol name, location, Easy Apply).
It uses webbrowser, pyautogui to automate the process
Steps:
1-Modify the parameter.def file to include the roles. Separated by commas. ie: "Java Developer,Python Developer,Java full stack"
2-Add a line on parameter.def file to include the companies names. Separated by commas. ie: "Codira,Digital Embrace,Code Runners"
3-Include a excel file in the work directory. Setup the name of excel file in the script "link-automate.py"
4-Run the script link-automate.py
5-See the magic.
